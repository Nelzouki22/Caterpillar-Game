# Caterpillar Game
مشروع لعبة الدودة (Caterpillar Game) باستخدام Python:

في هذا المشروع، نقوم بإنشاء لعبة بسيطة باستخدام مكتبة turtle في Python، مستوحاة من لعبة الدودة الشهيرة التي كانت موجودة على الهواتف المحمولة القديمة. في هذه اللعبة، يتحكم اللاعب بدودة تزداد طولًا كلما أكلت "ورقة" (طعام) تظهر بشكل عشوائي على الشاشة. الهدف من اللعبة هو جمع أكبر عدد ممكن من الأوراق لزيادة النقاط قبل أن تصطدم الدودة بجدران الشاشة.

الميزات الأساسية:

حركة الدودة: تتحرك الدودة للأمام بسرعة ثابتة، ويمكنك التحكم في حركة الدودة باستخدام مفاتيح لوحة المفاتيح.
الورقة: تظهر الورقة في مواقع عشوائية على الشاشة، ويجب على الدودة الوصول إليها لتناولها.
تحديث النقاط: يتم تحديث النقاط بشكل مستمر كلما أكلت الدودة ورقة، وتظهر النقاط على الشاشة.
رسالة "انتهت اللعبة": عند اصطدام الدودة بجدران الشاشة، تظهر رسالة "انتهت اللعبة" على الشاشة.
متطلبات المشروع:

Project Description in English
Caterpillar Game Project using Python:

In this project, we create a simple game using Python's turtle library, inspired by the classic caterpillar game found on old mobile phones. In this game, the player controls a caterpillar that grows longer each time it eats a "leaf" (food) that appears randomly on the screen. The goal is to collect as many leaves as possible to increase the score before the caterpillar hits the screen edges.

Key Features:

Caterpillar Movement: The caterpillar moves forward at a constant speed, and you can control its movement using keyboard keys.
Leaf: The leaf appears at random positions on the screen, and the caterpillar must reach it to eat it.
Score Update: The score is continuously updated each time the caterpillar eats a leaf, and the score is displayed on the screen.
Game Over Message: When the caterpillar hits the screen edges, a "Game Over!" message appears on the screen.
Project Requirements:

Python with the turtle library
Basic knowledge of Python programming
